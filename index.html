About

Code Challange: use HTML, CSS, and Javascript to read the Punk API and display it in an organized way.

Your goal is to implement a web solution to display the beers returned by the API in pages, like products on Amazon Website. Two minimum pieces of information must be shown about every beer: name and description.

Besides this, the website must provide a search engine, in which the user informs a beer name, and, instead of continuing to show all the beers, it will show only those filtered by the API.
